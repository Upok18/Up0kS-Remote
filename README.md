A lightweight remote control application for Windows PCs.

Control your computer from your phone over your local network.

> ⚠️ Currently in active development.

---

## Features

### Power
- Shutdown
- Restart

### Media
- Toggle Mute
- Volume Up
- Volume Down
- Play / Pause
- Next Track
- Previous Track

### System
- Device Information
- Ping
- LAN Discovery
- Password Authentication

---

## Requirements

- Windows 10/11
- Python 3.14+
- Android app (coming soon)
